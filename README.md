Project Title:Intelligent-ai-image-analyzer-with-contextual-insights-interaction

Description:This project is a web application built using Streamlit that allows users to upload an image and ask questions about it.
It uses the Google Gemini API to generate intelligent responses based on the image and user prompt.

Features:
Upload image (JPG, PNG)
Ask questions about the image
AI-generated responses
Simple and interactive UI
Real-time processing

Tech Stack:
Python
Streamlit
google-generativeai
Pillow
python-dotenv

Project Structure:
project-folder/
│── app.py
│── requirements.txt
│── .env
│── README.md

Installation & Setup:

1.Clone Repository
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name

2.Install Dependencies
pip install -r requirements.txt

3.Add API Key
Create .env file:
API_KEY=your_gemini_api_key

4.Run the App
streamlit run app.py

5.Deployment
This application is deployed on Streamlit Community Cloud and can be accessed from anywhere using a web browser.

steps to deploy:
    1.Push code to GitHub
    2️. Go to Streamlit Cloud
       Connect your GitHub repository
       Select app.py
    3️. Add Secrets
       In App Settings → Secrets:
       API_KEY = "your_gemini_api_key"
    4.Deploy
      Click Deploy and your app will be live
